JavaScript Notes

To open JavaScript Console in Chrome you can use:

  • Press Option + Command
  • Right-Click on the web page, Select Inspect Element, then click on the Console tab.

To comment use either // before single line comment

For multiple line comment you can use /* to open the comment and */ to close it.
